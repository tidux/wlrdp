# WLRDP

A multiuser, multisession, compositor-agnostic Wayland RDP Terminal Server.

## Status

WLRDP is alpha software, written in C with no guard rails.  It may contain RCEs or other fun 0days.  Patches welcome.

## History

I was inspired by [XRDP](https://xrdp.org), but I had seen them, along with TurboVNC, TightVNC, X2Go, and others, all struggle to make the switch from X.  Wayland and X11 are just too different, so a clean-sheet approach was needed.
